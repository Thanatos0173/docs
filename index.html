Utilisation

Il est possible de télécharger simplement le programme, sans rien avoir à compiler en téléchargant le fichier is. Pour tout compiler toutefois, il faut télécharger: - Le fichier makefile - Les fichiers dans /types - Les fichiers dans /CLI

Il est également nécessaire de suivre l'arborescence du repository.

Une fois tout installé, il suffit de taper la commande make pour générer l'exécutable is

Help menu

Le menu d'aide se trouve dans le fichier command_list.md, ou en tapant la commande ./is -h
