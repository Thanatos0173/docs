Ajoute les arêtes du graphe pour qu'il corresponde à l'image.

Parameters
imagePointeur vers l'image à segmenter
epsilonValeur qui définit à quel point la segmentation est précise
doVerboseIndique si le flag verbose vaut 1 ou 0
graphLe graphe qui va être modifié. Il est passé en argument afin de pouvoir récupérer des statistiques si nécessaire.
wishedTranformationDans quel color space les couleurs devraient être transformées. Voir conversion.h pour plus d'informations.
doRandomSi doRandom est mis à vrai, on ajoute les voisins dans un ordre aléatoire. Sinon, on les ajoute dans l'ordre suivant : voisin de droite, voisin de gauche, voisin du bas, voisin du haut
Return values
cc*Les composantes connexe du graphe représentant l'image
